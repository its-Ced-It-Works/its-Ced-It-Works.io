What have I worked on?


Dimension Data Glance

Dimension Data Glace came in fourth place at Dimension Data's Hackfest 2019.1. It is a webapp that uses the scanning of QR codes to enable frictionless ticket logging of configuration items in meeting rooms, like projectors, for Dimension Data employees.

Dimension Data Dash

Dimension Data Dash is an enablement tool for Dimension Data Security employees. It acts as an encylopedia of the various security controls the Security division deals with. It answers three key questions related to each control; What is it, Why's it important; What's the potential risk of not having it in place.
